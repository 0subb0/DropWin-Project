# DropWin - 경매형 중고 거래 플랫폼
"실시간 경매와 입찰 기능을 제공하는 중고거래 플랫폼"


주요 기능:
- 회원가입 / 로그인 / 아이디 중복 확인 / 아이디,비밀번호 찾기
- 상품 등록 / 조회 / 입찰 기능
- 나의 입찰 현황 확인
- 공지사항 게시판 및 검색 기능

기술 스택:
- Backend: Spring Boot, MyBatis
- Frontend: HTML/CSS/JS, Thymeleaf
- Database: MySQL
